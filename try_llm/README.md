# INSTALL
```console
sudo apt install python3-virtualenv
pip install --upgrade virtualenv
virtualenv -p python3 venv
chmod +x venv/bin/activate
```

Activate python venv
```console
source ./venv/bin/activate
```

# ENVIRONEMENT

```console

echo "export OPENAI_API_KEY='yourkey'" >> ~/.zshrc

```
